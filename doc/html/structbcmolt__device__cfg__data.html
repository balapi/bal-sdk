bcmos_bool wd_enable

In PCIe out of band management, this field is ignored. For in-band systems, the default is for the watchdog to be enabled. When enabled the Maple watchdog is patted periodically.

uint16_t wd_ip_traffic_timeout

The units are seconds.

The watchdog checks for IP traffic periodically.

The watchdog IP traffic check default period is 5 minutes (300 seconds).

The watchdog IP traffic check period minimum has to be greater than the product of ka interval and (ka tolerance + 1).

The minimum value of ka interval and ka tolerance is 0. In that case the minimum comm check period minimum is 5 seconds.

If no IP traffic is received within the configured period. The watchdog will reset the Maple chip.
