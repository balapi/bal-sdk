Field Documentation

uint8_t re

Indicates whether the Transmit Optical Level (TOL) contains the launch power of the OTL (RE=0) or of a reach extender (RE=1)

bcmolt_odn_class odn_class

Identifies the nominal optical parameters of the transceiver according to the ODN Optical Path Loss (OPL)

bcmolt_control_state ds_fec_mode

Enable/Disable the downstream FEC. Default is Enable. Attribute can be set only when PON NI state is Inactive.

bcmolt_tc_protocol protocol

When system mode is NGPON2, this parameter Indicate the TC layer protocol: ITU-T G.989.3 or ITU-T G.987.3

bcmolt_link_type ds_link_type

Optical link type (Unspecified, A, B, Both)

bcmolt_pon_id pon_id

Identifies the TWDM channel termination within a certain domain

uint8_t c

Transmit optical level (TOL) reference point indicator: S/R-CG or S/R-CP

uint16_t tol

Transmit Optical Level. An indication of the current OLT CT transceiver channel launch power into the ODN
