Field Documentation

bcmolt_obj_id obj_type

Object type. Can be BCMOLT_OBJECT_ANY

f_bcmolt_msg_handler rx_cb

Receive callback. Must be != NULL

bcmolt_auto_flags flags

Dispatch flags. Receive callback function can be called in the context of rx task or application task, depending on the flags

bcmos_module_id module

Target module ID. Relevant only if flags == BCMOLT_AUTO_FLAGS_DISPATCH

uint32_t pon_ni_mask

Bitmask of pon_ni interfaces the registration applies to. 0=all interfaces. Each interface is identified by (1 << pon_ni) bit

uint16_t subgroup

If set, registration is performed for specific object's subgroup
