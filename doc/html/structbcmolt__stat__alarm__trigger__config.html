◆ rising

uint64_t rising

The alarm is raised if the stats delta value per second becomes greater than this threshold level.

◆ falling

uint64_t falling

The alarm is cleared if the stats delta value per second becomes less than this threshold level.

◆ upper

uint64_t upper

The alarm is raised if the stats delta value per second becomes greater than this upper level.

◆ lower

uint64_t lower

The alarm is raised if the stats delta value per second becomes less than this lower level.

◆ limit

uint64_t limit

The alarm is raised if the stats sample value becomes greater than this level. The alarm is cleared when the host clears the stats.
